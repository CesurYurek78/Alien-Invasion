🎮 Mein Spiel – Alien Invasion
Ein einfaches, aber voll funktionsfähiges Browser-Spiel, das als Grundgerüst für eigene Spieleprojekte dient. Es demonstriert die grundlegenden Konzepte der Spieleentwicklung mit HTML5 Canvas, JavaScript und CSS – inklusive Menüsteuerung, Zustandsmanagement und einem simulierten Punktesystem.

🌟 Eigenschaften
Hauptmenü – Startbildschirm mit Start-Button

Spielzustände – saubere Trennung zwischen Menü- und Spielmodus

Canvas-Rendering – dynamisches Zeichnen von Himmel, Gras, Spieler und Punktestand

Reines HTML/JS/CSS – keine externen Bibliotheken oder Abhängigkeiten

🕹️ Spielmechanik
Das Spiel ist bewusst einfach gehalten, um als Lernbeispiel zu dienen:

Im Menü klickst du auf Start.

Oben sind Aliens und die muss man mit Space Taste abschießen. 

Bei Controls werden die Steuerungen erklärt.

Alien Sachbuch erklärt die Eigenschaften von den Aliens

Shop erlaubt den Spieler die eingesammelten Punkte zu benutzen

Wenn man den Endgegner besiegt hat, kann man sich die Extra Levels anschauen.

🛠️ Verwendete Technologien
HTML5 – Struktur und Canvas-Element

CSS3 – Styling, Overlay, Buttons

Canvas API – 2D-Zeichenfunktionen

🚀 So startest du das Spiel
Lade den gesamten Code als .html-Datei herunter (z.B. spiel.html).

Öffne die Datei in einem modernen Browser (Chrome, Firefox, Edge, Safari).

Klicke auf den Start-Button und beobachte, wie die Punkte steigen.

📁 Projektstruktur
Die gesamte Anwendung besteht aus einer einzigen HTML-Datei, die alle Styles und Skripte enthält – ideal für den Einstieg oder für schnelle Prototypen.

Dieses Spiel kann man auf Windows, Mac und auf Linux spielen, weil das ein Browser Spiel ist.

Wenn man das Spiel schließt und wieder öffnet, verliert man seine Fortschritte im Spiel nicht.
